Xray Web 管理面板

x-ui

基于 Go 构建的轻量级 Xray 管理面板，浏览器管理入站代理、用户配置、流量统计，一站式搞定。

Go + Gin Xray-core MIT 开源
立即安装 查看 GitHub
Features

为什么选择 x-ui？

Linux 服务器上管理 Xray 代理所需的一切

Web 可视化管理

浏览器直接管理 Xray 入站配置，支持多用户、多协议、多传输方式。

系统状态监控

实时查看 CPU、内存、磁盘、网络流量、TCP/UDP 连接数等运行状态。

安全加固

CSRF 双层防护、登录限速、用户隔离、TG Token 加密存储。

Telegram 通知

流量统计、登录提醒、到期提醒、流量预警，通过 Telegram Bot 实时推送。

CGO-Free 构建

纯 Go SQLite 驱动，无需 CGO，跨平台编译更简单。

v2-ui 迁移

一键迁移 v2-ui 的入站配置和用户数据，平滑升级。

Protocols

支持协议

覆盖主流代理协议，满足不同场景需求

VLESS

VLESS + Reality

无需域名和证书，抗检测能力最强

VMess

VMess + WS + TLS

客户端兼容性最好，CDN 中转友好

Trojan

Trojan + TLS

配置简单，基于 TLS 伪装性好

SS

Shadowsocks

经典轻量代理，支持 AEAD 加密

SOCKS

SOCKS5

标准代理协议，支持 UDP 转发

HTTP

HTTP 代理

标准 HTTP 代理，支持用户认证

Door

Dokodemo-door

透明代理和端口转发
