x-ui

基于 Go + Gin + Xray-core 的轻量级 Web 管理面板，面向 Linux 服务器部署，提供 Xray 入站配置、用户管理、流量统计一站式管理。

立即安装 下载 Release GitHub

项目特性

开箱即用的 Xray 代理面板管理体验

🕸

Web 可视化管理

浏览器直接管理 Xray 入站配置，支持多用户、多协议、多传输方式。

📊

系统状态监控

实时查看 CPU、内存、磁盘、网络流量、TCP/UDP 连接数等运行状态。

🔒

安全加固

CSRF 双层防护、登录限速、Xray 用户隔离、Systemd 安全指令、TG Token 加密存储。

🤖

Telegram 通知

流量统计、登录提醒、到期提醒、流量预警，通过 Telegram Bot 实时推送。

📦

CGO-Free 构建

纯 Go SQLite 驱动，无需 CGO 即可编译，跨平台构建更简单。

🔄

v2-ui 迁移

一键迁移 v2-ui 的 inbound 账号数据，平滑升级到 x-ui。

支持协议

覆盖主流代理协议，满足不同场景需求

VLESS

VLESS + Reality

无需域名和证书，抗检测能力最强，首选方案

VMess

VMess + WS + TLS

经典方案，客户端兼容性最好，CDN 中转友好

Trojan

Trojan + TLS

配置简单，协议本身基于 TLS，伪装性好

Shadowsocks

Shadowsocks

经典轻量代理协议，兼容性最好，支持 AEAD 加密

SOCKS

SOCKS5

标准代理协议，支持 UDP 转发和用户名密码认证

HTTP

HTTP 代理

标准 HTTP 代理协议，支持用户名密码认证

Dokodemo

Dokodemo-door

任意门转发，透明代理和端口转发场景

一键安装

在 Debian / Ubuntu / CentOS 服务器上快速部署

$ bash <(curl -Ls https://raw.githubusercontent.com/motao123/x-ui/main/install.sh)

安装后常用管理命令：

x-ui管理菜单
x-ui status查看状态
x-ui start启动面板
x-ui stop停止面板
x-ui restart重启面板
x-ui log查看日志
x-ui update更新面板
x-ui uninstall卸载面板
