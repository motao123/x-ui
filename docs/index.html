Features

Why x-ui?

Everything you need to manage Xray proxies on a Linux server

Web Management

Configure Xray inbounds, users, and protocols through an intuitive browser-based UI.

System Monitoring

Real-time CPU, memory, disk, network usage with TCP/UDP connection tracking.

Security Hardening

Dual-layer CSRF protection, login rate limiting, user isolation, encrypted TG tokens.

Telegram Alerts

Traffic stats, login alerts, expiry reminders, and usage warnings via Telegram Bot.

CGO-Free Build

Pure Go SQLite driver, no CGO required. Cross-compile for any platform easily.

v2-ui Migration

One-click migration from v2-ui. Import existing inbound configs and user data.
